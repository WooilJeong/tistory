## tistory

**tistory**는 티스토리 블로그 자동화를 위한 오픈소스 파이썬 라이브러리입니다. 티스토리에서 제공하는 Open API를 Python으로 쉽게 이용할 수 있습니다.


## 설치 방법

```bash
pip install tistory
```
